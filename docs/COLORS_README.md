## Usage Guidelines

- **Primary colors** are used for main actions and important elements
- **Success colors** indicate positive actions or successful states
- **Warning colors** indicate caution or pending states
- **Destructive colors** indicate dangerous actions or error states
- **Info colors** are used for informational messages
- **Risk colors** are used for risk assessment visualization
- **Chart colors** provide a consistent palette for data visualization
- **Sidebar colors** ensure proper contrast and hierarchy in navigation

All colors follow accessibility guidelines for contrast ratios.
